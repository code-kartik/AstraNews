# Unsplash Photo Explorer App

## Overview
This Android application serves as a demonstration of implementing Explore and Refine screens using the Unsplash API. The app allows users to explore and refine a collection of photos from Unsplash.

## Features
- **Explore Screen:** Displays a collection of photos fetched from the Unsplash API.
- **Refine Screen:** Enables users to apply filters to refine the displayed photos based on categories like Wallpapers, 3D Renders, Nature, and more.
- **Search Functionality:** Users can search for photos based on keywords using the search feature in the Refine Screen.

## Demo and Limitations
- This app is a demo and should not be considered a fully functional application.
- It was created to fulfill specific tasks related to the implementation of Explore and Refine Screens.
- The app lacks certain features and robustness that a production-ready application would have.
- Use it as a reference or starting point for further development rather than as a complete, standalone application.

## How to Use
1. Install the app on an Android device or emulator.
2. Explore Screen: View a collection of photos.
3. Refine Screen: Apply filters to refine the displayed photos or use the search feature.
4. Observe the behavior and interaction of the app.
